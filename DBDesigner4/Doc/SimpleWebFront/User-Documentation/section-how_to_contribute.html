How to contribute?

if you like SWF and you would like to help the project, here are some ways you can contribute:

  • Send in good bug reports. Either mail them to <ulli (at) fabforce.net> or report them on the forum.

  • Provide useful feedback and suggestions.

  • If you are an English native speaker, please report grammatical mistakes in this documentation and elsewhere. Or even better send me a corrected version of this document. Preferably as a patch against the xml-version of this document.

  • Improve the PHP-code of the templates or write your own layout. I would be happy to publish templates with different designs or features. Read the developer documentation for more information on that.

  • The php-code still lacks the feature to search for data records.
