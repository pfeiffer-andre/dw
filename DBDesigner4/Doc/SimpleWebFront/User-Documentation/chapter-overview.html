Introduction

The SimpleWebFront (abbreviated SWF in the following) is a plugin for the fabFORCE DBDesigner4 database modeler which can create web-pages for managing the tables defined in your DBD4-model. The created webpages are realized as PHP-code. Apart from showing the contents of a db-table you can edit, insert and delete date records.

SimpleWebFront is an Open Source Project available for Microsoft Windows© 2k/XP and Linux KDE/Gnome. It is released under the GNU GPL.
