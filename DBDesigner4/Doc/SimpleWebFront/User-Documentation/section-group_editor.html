The Group Editor

Use the group editor to set the attributes of a group.

Figure 2-6. Group Editor

  • The Group-Name is used to refer to this specific group in all situations. (in the SWF-program as well as in the generated webpages)

  • The Show Views in a Popup-Window flag determines if the Grid-View for all views of this group opens in the same or in a popup-window when the user clicks on the respective link on the index-site. if this option is checked the popup-screen-position can also be specified for each view. See the Section called Grid Options.

  • It is also possible to specify the positon of this group on the index-webpage. This is done in the placement section. Imagine that the index-webpage uses one big table for layouting the group position. You can now specify the row and colum-number of each group.
