Views

This tab lets you create views. Please see the Section called Basic Concepts in Chapter 1 for an explanation of views.

Figure 2-3. Views Tab

On the left hand side there is the view management section. It consists of a listbox which will show all available views, and four view management buttons.

The listbox will show all views that have been created, regardless whether they have been assigned to a group or not.

  • Clicking the Create View-Button it the normal way to create a new view. After clicking the button the View-Editor-Dialog is showed.

  • Clicking the Delete View-Button will delete the currently selected view in the list.

  • By clicking the Edit View-Button you have the possibility to edit an already created view. Unsurprisingly the currently selected view in the listbox will be edited.

  • The Add Tables-Button provides a way to conveniently add all tables of the model as views. This means that for each table in the model a view is created and a created view will just show its Main-Table. (no where-clause and no nm or lookup-tables are specified).

The View information box on the right side will show you some data about the currently selected view in the listbox.
