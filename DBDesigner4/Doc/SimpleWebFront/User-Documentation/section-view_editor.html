The View Editor

The view editor is the place where you can enter/edit the most basic settings of a view.

Figure 2-4. View Editor

  • The viewname is used to refer to this specific view in all situations. (in the SWF-program as well as in the generated webpages)

  • The table select-field lets you choose the Main-Table. Notice that the columns of the currently selected table are displayed below in the "Table Information"-section.

  • You can also choose an icon for the view. This icon is displayed on the index-webpage along with the viewname. To select an icon you have to double-click the currently selected icon. You can either choose one of the standard-icons that are distributed with the program or one of your own pictures.

  • The table information section does not only list the columns of the view but also allows you to define the joins that should be made.

    As soon as you have selected one table for a join (no matter if it is a lookup or nm-table join) the columns of the joined table are added to columns-list of the view.

  • At the bottom of the dialog you can finally specify a Where-Clause for your view.
